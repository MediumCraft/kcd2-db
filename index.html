Welcome to the Kingdom Come: Deliverance II Item Database

This website can find the item ID of any item in the game.


The site is not designed for mobile devices and uses a lot (~500 MB) of memory.

You are currently running on the static build of the website, which has worse performance than the live site.

As of right now, the live site is not public.

Please ignore the summarize button in documents, as well as other community features. Those only work on the live version.

Also apologies for the bad site design and performance issues.


You can search with the Name, ID and the Type. Searching with items in other languages should work, but it hasn't been tested.

If a value is highlighted, click to copy it.


Append @ to an item type to only search through that specific item type. Use !@ to exclude a specific item type.

Append # to search through specific fields of an item.

The search bar should automatically provide autocomplete.
